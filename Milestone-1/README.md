# AI Skill Gap Analyzer

A Streamlit-based app to analyze resumes and job descriptions, highlighting skill gaps.

## Features
- Upload resumes and job descriptions (PDF, DOCX, TXT)
- Extract and clean text
- Analyze and compare documents
- Download processed results (CSV/JSON)

## How to Run
1. Install requirements:  
   `pip install streamlit pandas`
2. Start the app:  
   `streamlit run main_app.py`

## Folder Structure
- `src/` - Document processing and pipeline scripts
- `main_app.py` - Streamlit UI
