# Resume_Analyzer
SkillGapAI: Analyzing Resume and Job Post for Skill Gap_Metting_Invite

## 📌 Project Overview
SkillGapAI processes resumes and job postings to extract useful information such as text, skills, and structure.  
It identifies skill gaps between a candidate’s resume and the requirements of a job description, helping candidates align their profiles with job opportunities.

---

## ✨ Features
- Upload and process resumes in **PDF, DOCX, and TXT** formats  
- Extract raw text content from resumes and job descriptions  
- Clean and preprocess extracted data  
- Analyze resume structure and extract skills  
- Compare with job posting skills to find **skill gaps**  
- Save processed text for further use  

---

## 🛠 Tech Stack
- **Python 3.x**  
- **VS Code**  
- **Git & GitHub**  
- Libraries:
  - `pdfplumber` (for PDF reading)  
  - `python-docx` (for DOCX reading)
